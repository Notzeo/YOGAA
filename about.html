About Yogasthali

Hey there! Welcome to Yogasthali, developed in 2023.

We created Yogasthali inspired by various Machine Learning models for visual recognition. This is our first Machine Learning model and our first attempt at Web Development. we have loved (almost) every second of this project and we have learnt more than we could have imagined.

Yogasthali uses Pose.Net, developed by Google’s TensorFlow, and ml5.js, a JavaScript library for Machine Learning on the browser. Most of our learning was following The Coding Train’s YouTube playlist on Pose Estimation.
